# **App Name**: SalonVerse

## Core Features:

- Salon Discovery: Public users can filter salons by state and city, and view salon cards with name, landmark, and service menu.
- Appointment Request: Public users can request appointments through the app.
- Salon Owner Registration: Salon owners can register their shop details, set a manual address/landmark, and manage bookings.
- Subscription Management: Handle monthly subscription payments and toggle salon visibility based on payment status.
- Booking Management: Salon owners can accept or reject booking requests, triggering SMS notifications to users with landmark & address upon acceptance, or nearby alternatives upon rejection.
- No-Show Flagging: Salon owners can flag users who don't show up for appointments. The app should ban users with excessive flags.
- Admin Dashboard: Admins can verify new salon registrations, track revenue, and manually override salon visibility.

## Style Guidelines:

- Primary color: Deep sky blue (#32ade6) to convey trust and professionalism.
- Background color: Light gray (#f0f2f5), providing a clean, neutral backdrop.
- Accent color: Soft orange (#f0ab3d) for a warm contrast in key interactive elements such as the primary call to action.
- Body and headline font: 'Inter', sans-serif, providing a modern, machined, objective and neutral look.
- Use clean, professional icons to represent salon services and actions.
- Maintain a clear, card-based layout for salon listings, emphasizing key information.
- Use subtle transitions and loading animations to enhance user experience.
